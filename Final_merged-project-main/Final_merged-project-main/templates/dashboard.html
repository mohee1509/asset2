Steps to Get Started

  1. Ensure you have Python 3.8+ installed.
  2. Install required agent dependencies: 
    pip install psutil requests
  3. Download the agent for your OS from above.
  4. Run the agent from your terminal (e.g., for Windows): 
    python win_agent.py
  5. The agent will start sending heartbeats to the server. View its status on the Server Dashboard.
